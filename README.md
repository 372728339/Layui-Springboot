# Layui-Springboot
## 项目简介
  - 后端使用springboot+mybatis
  - 前端使用原生js和layui
  - 数据库采用mysql
## 使用方法
1. 导入mhzxhoutai.sql至mysql
2. 打开工程
3. 修改src/main/resources/application.yml中的数据库密码以及端口
4. 运行项目，成功后访问http://127.0.0.1:端口号
5. 测试最高权限账号：3213446 密码： 3213446
